## 使用前必需的配置（必看）
### 需要为随问打开辅助功能授权和屏幕录制权限
由于随问需要获取用户划选的文本和用户的屏幕进而实现文本识别，需要用户打开这两个功能，若没有打开这两个授权，随问会提示用户进行打开

### 需要为随问提供大模型API
**随问目前本身不提供大模型服务，需要用户自己申请各大模型的API并将Base Url、API密钥、模型名称提供给随问，（有了这三个项，随问就可以调用对应的大模型为你提供服务啦）。**
在菜单栏中找到随问的图标，点击后打开配置，将申请到的Base Url、API密钥、模型名称填入如下的空白处
![配置](/static/images/Setting.png)
API的申请教程非常简单，这里推荐使用火山引擎的DeepSeek V3 API（作者就在使用这个），新用户会提供50万token数，可以用非常久。API与订阅制的大模型服务不同，属于按量付费，所以非常省钱。

具体多划算呢，通常1个中文词语、英文单词、数字、符号计为 1 个token。拿DeepSeek举例，一百万输入和输出Token都在10块钱以内，也就是说，花费不到十块钱就能处理和输出共计100万词语的文本。

**火山引擎的DeepSeek API申请详细教程已经放在了第二篇文章中，申请过程超快超简单！**

## 如何使用
配置完大模型后，点击测试连接按钮，若出现成功的提示则证明大模型服务已经连通，此时已经可以使用随问
![配置成功](/static/images/ConfigSuccess.png)

随问提供划选唤起与截屏唤起两种功能
### 鼠标划选唤起随问（默认快捷键为：`⌥ + E`）
鼠标划选需要提供给随问的文本，然后按下划选唤起的快捷键，便会在屏幕上唤起随问为你提供服务。

### 截屏选取文本唤起随问（默认快捷键为：`⌥ + W`）
对于无法使用鼠标划选的文本，比如图片，PDF等，随问支持截屏并使用OCR识别文本并唤起随问。

### 随问提供的服务
**若你提供给随问的文本是一个专有概念或是有任何难以理解的东西，或是你看不懂这段话的意思，随问会自动识别你的意图并为你详细解释你划选的文本**
![划选解释](/static/images/SelectExplanation.png)
**若你提供的文本是外文单词或是句子，随问会为你提供详细的翻译和例句**
![划选翻译](/static/images/Translate.png)
**若你提供的文本是长段的文本，随问会为你做出总结**
使用TIPS：直接使用`⌘ + A`全选网页、文章、微信公众号文章等。然后使用鼠标划选唤起快解键唤醒随问，即可得到随问的总结。
![划选总结](/static/images/Summary.png)

### 多轮问答
对于尚未解释清楚的回答，你可以在随问的输入框中输入更进一步的问题来追问随问，其中随问可看见的最大上文条数在设置中可以设置，默认为6条。
