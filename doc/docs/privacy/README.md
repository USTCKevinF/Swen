## 隐私条款
随问 是一款 macOS 平台的解释/翻译/总结和 OCR 软件，主要通过集成多家第三方大模型服务商的服务完成需求。

## 隐私权限
为了让程序功能正常运行，需要为随问开启下列隐私权限：

- 辅助功能权限：通过辅助功能权限获取当前选中的文本，用于实现划词解释相关功能，当您使用到相关功能时随问才会向您请求该权限。
- 屏幕录制权限：通过屏幕录制权限获取指定区域的图像，用于识别其中的文本，截屏唤起随问需要使用该功能，当且仅当您使用到相关功能时随问才会向您请求该权限。
- 自动化权限：通过自动化权限控制浏览器用以获取当前选中的文本，当且仅当您使用到相关功能时随问才会向您请求该权限。

## 数据收集
随问的所有服务都运行在用户本地，所有的数据也由本地存储和处理，用户提供给随问的文本，随问也会在本地发起网络请求来通过第三方大模型服务商来获取回复。

## 本隐私政策的变更
我们可能会不时地更新我们的隐私政策以适应随问的最新情况和更新。因此，我们建议您定期查看本页面以了解最新的内容。
